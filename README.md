# Jiaye-Skills

> Product manager skill collection for AI agents — competitive benchmarking and PRD review simulation, each outputting professional HTML reports.

## Skills

| Skill | Directory | What It Does |
|-------|-----------|--------------|
| **Competitive Product Research** | `competitive-product-research/` | 8-dimension UX benchmarking → HTML report with S/A/B/C health score, reusable patterns, P0/P1/P2 recommendations |
| **PRD Review Simulator** | `requirement-review-simulator/` | 5-role cross-functional stress-test → HTML survival report with radar chart, counterargument playbook, meeting script |

## Quick Start

```text
# Competitive research
Our app's post conversion rate is only 3%. Compare with Xiaohongshu and Instagram.

# PRD review
Review my group-buying feature PRD. Realistic mode.
```

## Install

```bash
# Install individual skills from ClawHub
npx clawhub@latest install competitive-product-research
npx clawhub@latest install requirement-review-simulator
```

# 中文说明

## 技能列表

| 技能 | 目录 | 主要能力 |
|------|------|----------|
| **竞品调研** | `competitive-product-research/` | 8 维度结构化对标分析，输出带健康度评分和落地建议的 HTML 报告 |
| **需求评审模拟器** | `requirement-review-simulator/` | 五方攻防推演、PRD 存活率评分、三级残酷度、HTML 评审报告 |

```text
帮我评审一下我们要做的拼团功能，实战模式。
```
